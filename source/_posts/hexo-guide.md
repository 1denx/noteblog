## 快速開始
<!--more-->
### 初始化一個新的 Hexo 部落格
``` bash
$ hexo init 你的網站名稱
```
### 啟動本地伺服器，預覽網站
``` bash
$ hexo server
```

### 新建文章
``` bash
$ hexo new "文章標題"
```

### 生成靜態網頁文件
``` bash
$ hexo generate
```

### 清除快取和暫存文件
``` bash
$ hexo clean
```

### 部署網站至遠端伺服器
``` bash
$ hexo deploy
```
